## 컴포넌트 상세

### 1. Hero 섹션
- 그래디언트 배경 (노란색 카카오 계열)
- `.hero-badge`: 브랜드 뱃지
- `.hero-top`: 타이틀 영역
- `.hero-action`: CTA 버튼 ("새 앰배서더 등록")

### 2. 필터 바
- `.controls`: 컨트롤 영역 래퍼
- `.filter-group`: 필터 버튼 그룹
- `.filter-btn`: 개별 필터 버튼
  - 활성화 시: 역할별 색상 배경 적용
  - 비활성: 기본 스타일

### 3. 앰배서더 카드
```
┌────────────────────┐
│  [수정]   [뱃지]   │  .card-head
│   🟡 아바타         │  .avatar (역할별 색상)
│                    │
│  이름              │  h3
│  한 줄 소개...      │  .bio (2줄 말줄임)
│  [소개 …]          │  토글 버튼 (38자 초과 시)
│                    │
│  📷 💼 📝 🐦       │  SNS 아이콘 링크
└────────────────────┘
```

**카드 그리드**:
```css
grid-template-columns: repeat(auto-fill, minmax(220px, 1fr));
```

### 4. 등록/수정 모달
- `.modal-backdrop`: 반투명 오버레이
- `.modal-panel`: 편집 폼 패널 (2열 그리드)
- 폼 필드: 이름, 역할 선택, 한 줄 소개, SNS 9종
- 버튼: 저장, 초기화, 삭제 (수정 모드에서만 삭제 표시)
- 모달 제목: 신규 → "새 앰배서더 등록" / 수정 → "앰배서더 수정"

## 반응형 브레이크포인트

| 화면 | 조건 | 카드 그리드 | 필터 |
|------|------|-----------|------|
| 데스크톱 | 기본 | `auto-fill, minmax(220px, 1fr)` | 가로 나열 |
| 태블릿 | `≤ 768px` | 1열 | 가로 스크롤 |
| 모바일 | `≤ 420px` | 1열 | 2열 그리드 |

## 인터랙션 패턴

### 바이오 토글
- 38자 이하: 전문 표시
- 38자 초과: 2줄 말줄임 + "소개 …" 버튼 → 클릭 시 전문 펼침

### 카드 수정
- 카드 "수정" 버튼 클릭 → 모달 열림 (기존 데이터 채워진 상태)
- 모달에서 수정 후 "저장" → PUT API 호출

### 필터 전환
- 필터 버튼 클릭 → 해당 역할 카드만 표시
- "전체" 클릭 → 모든 카드 표시
- 검색과 필터 동시 적용 가능
